# Open edX Docker

## Set Enverionment and Build

```
$ export EDX_RELEASE="lilac.1"
$ export FLAVOR="bare"
$ export EDX_RELEASE_REF="open-release/lilac.1"
$ export EDX_DEMO_RELEASE_REF="open-release/lilac.1"
$ make bootstrap
```
